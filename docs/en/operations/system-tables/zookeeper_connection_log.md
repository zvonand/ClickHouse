# system.zookeeper_connection_log

<SystemTableCloud/>

The 'system.zookeeper_connection_log' table shows the history of ZooKeeper connections (including auxiliary ZooKeepers). Each row shows information about one event regarding connections.

:::note
The table doesn't contain events for disconnections caused by server shutdown.
:::

Columns:


<!--AUTOGENERATED_START-->
- `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Hostname of the server which is connected to or disconnected from ZooKeeper.
- `type` ([Enum8(\'Connected\' = 0, \'Disconnected\' = 1)](/sql-reference/data-types/enum)) — The type of the event. Possible values: Connected, Disconnected.
- `event_date` ([Date](/sql-reference/data-types/date)) — Date of the entry.
- `event_time` ([DateTime](/sql-reference/data-types/datetime)) — Time of the entry
- `event_time_microseconds` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — Time of the entry with microseconds precision.
- `name` ([String](/sql-reference/data-types/string)) — ZooKeeper cluster\'s name.
- `host` ([String](/sql-reference/data-types/string)) — The hostname/IP of the ZooKeeper node that ClickHouse connected to or disconnected from.
- `port` ([UInt16](/sql-reference/data-types/int-uint)) — The port of the ZooKeeper node that ClickHouse connected to or disconnected from.
- `index` ([UInt8](/sql-reference/data-types/int-uint)) — The index of the ZooKeeper node that ClickHouse connected to or disconnected from. The index is from ZooKeeper config.
- `client_id` ([Int64](/sql-reference/data-types/int-uint)) — Session id of the connection.

Only applicable to ClickHouse Keeper.
- `availability_zone` ([String](/sql-reference/data-types/string)) — Availability zone
- `reason` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Reason for the connection or disconnection.
